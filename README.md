To Do list application

A simple, beginner-friendly Python project

1. Project Title

Simple To-Do List Application


2. Overview of the Project

This project is a basic command-line To-Do List application built using Python.
It allows users to add tasks, view tasks, and exit the program.
It is ideal for beginners learning loops, lists, and conditional statements.


3. Features
	•	Add a new task
	•	View all added tasks
	•	Tasks stored in a Python list
	•	Simple and clean menu
	•	Beginner-friendly code
	•	No external libraries required


4. Technologies / Tools Used
	•	Python 3.x
	•	Runs in any terminal/IDE (IDLE, VS Code, PyCharm)


5. Steps to Install & Run the Project
	1.	Install Python on your system.
	2.	Copy the Python code into a file named todo.py.
	3.	Open terminal and run:
       python todo.py
  4.	Use the menu to add/view tasks.


6. Instructions for Testing
	1.	Start the program
	2.	Choose Option 1 → Add multiple tasks
	3.	Choose Option 2 → Confirm the tasks are displayed correctly
	4.	Choose Option 3 → Check if the program exits properly
